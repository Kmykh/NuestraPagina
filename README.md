# NuestraPágina

Aplicación React creada con Vite que mantiene la temática romántica de "Ratón & Ratoncita" con animaciones, pantalla de acceso y secciones interactivas.

## Requisitos previos
- Node.js 18 o superior
- npm 9 o superior

## Scripts disponibles
- `npm install`: instala dependencias.
- `npm run dev`: inicia el entorno de desarrollo en http://localhost:5173.
- `npm run build`: genera la versión optimizada dentro de `dist/`.
- `npm run preview`: sirve el build generado para revisión final.

## Notas
- Coloca tus imágenes (`foto1.jpeg`, `foto2.jpeg`, `foto3.png`) dentro de la carpeta `public/` (o en la raíz del proyecto) para que Vite pueda servirlas.
- El código secreto sigue siendo **2211** y el estado "intro vista" se guarda en `localStorage`.
